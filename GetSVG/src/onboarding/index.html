How to use

Get started in seconds

1

Pin the extension

Click the puzzle icon in Chrome toolbar, then pin GetSVG. Keeps it one click away on every tab.

2

Highlight All SVGs

Flash a bright yellow border around every SVG on the page. Toggle it off just as easily. Find hidden icons and assets you didn't know were there.

3

Right-click → Copy

Right-click any SVG on the page to instantly copy the clean SVG markup to clipboard. Works on inline SVGs, image tags, and background SVGs.

4

Right-click → Save

Download any SVG directly to your filesystem with one click from the context menu. File is cleaned and ready to use.

5

Browse All SVGs in Panel

Open the extension popup to see every SVG on the current page. Copy, download, or locate each one from a clean thumbnail list.
